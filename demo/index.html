LightSpeedBox

Lightbox plugin for jQuery

About

LightSpeedBox is a jQuery plugin for organization of a fancy preview box for images (also usually called a lightbox).

Key features:

  • Show the lightbox by click on a thumbnail
  • Support displaying individual images as well as switching through a set
  • Controls: next, previous image, close button, keyboard, swipe for touch screens
  • Smooth transitions and animations
  • Localization
